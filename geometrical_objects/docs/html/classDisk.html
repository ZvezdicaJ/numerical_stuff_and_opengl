Detailed Description

template<typename T = float>
class Disk< T >

A class holding vertexes in the shape of a disk.

A class holding vertexes in the shape of a disk. Disk is a 3d shape, thus it inherits from Shape3D class. Template parameter can either be float or double.

void Disk< float >::generate_vertexes ( )
inlineprivate

Function which generates vertexes for float version of this class.

Vertexes are generate in such a way that the middle of the shape is (0,0,0). The function also sets the element array. It uses sse instructions - use appropriate processor.
