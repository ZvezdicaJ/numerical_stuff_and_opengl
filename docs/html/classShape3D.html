Detailed Description

template<typename T>
class Shape3D< T >

This is a base class for all 3D shapes.

It inherits from Shape class. Template parameter can either be float of double. It is meant to contain functions common to all 3D shapes - like area and volume.

This function calculates area of the given 3D shape.

It sums the areas of all triangles. For each triangle it calculates the area using Heron formula.
